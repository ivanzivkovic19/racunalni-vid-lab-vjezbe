# Računalni vid - Laboratorijske vježbe

Ovaj repozitorij sadrži riješene laboratorijske vježbe iz kolegija **Računalni vid**.

## Napomena

Tijekom izrade laboratorijskih vježbi imao sam problema s automatskim spremanjem i slanjem (pushanjem) datoteka na GitHub putem Google Colaba. Zbog toga neke datoteke nisu bile automatski ažurirane u repozitoriju.

Kako bi repozitorij sadržavao konačne verzije rješenja, u odgovarajućim mapama nalaze se laboratorijske vježbe sa zamijenjenim i dovršenim `.ipynb` datotekama koje predstavljaju završne verzije predanih zadataka.

## Sadržaj

Repozitorij sadrži:

* Laboratorijsku vježbu 1
* Laboratorijsku vježbu 2
* Laboratorijsku vježbu 3
* Laboratorijsku vježbu 4
* Laboratorijsku vježbu 5
* Laboratorijsku vježbu 6

Svaka mapa sadrži pripadajuću Jupyter Notebook (`.ipynb`) datoteku s riješenim zadacima.
